Sync / Cross-Device Transfer跨機器同步

Export your scenarios + watchlist as JSON, transfer to another machine, then import將場景與收藏清單匯出成 JSON，傳到另一台機器後再匯入

Why this page exists: Browser localStorage is device-local — data saved on your Mac mini is invisible on your MacBook. Workflow: Mac mini Export → transfer JSON via AirDrop / iCloud Drive / Email → MacBook Import. Language preference is not synced (stays per-device). 為什麼需要這頁： 瀏覽器 localStorage 只存在當前機器，Mac mini 儲存的資料在 MacBook 上看不到。作業流程：Mac mini 匯出 → AirDrop / iCloud Drive / Email 傳 JSON → MacBook 匯入。語言偏好不會被同步（各機器個別設定）。

Permanently removes all sync-able data from this machine's browser storage. Cannot be undone. 從本機瀏覽器儲存永久移除所有可同步資料。此操作無法還原。

Data stays local. Nothing is sent to any server — all export/import happens entirely in your browser. The JSON file is generated and downloaded client-side. 資料保持在本機。沒有任何資料傳送到伺服器，所有匯出／匯入操作皆在瀏覽器端完成。JSON 檔案在用戶端產生並下載。
