§1｜市場定位——RM774 psf 大坪豪宅的特殊邏輯

1.1 大坪 PSF 折扣的結構性解釋

MK28 的 RM774 psf（brickz 成交中位）看似低於同地段精品建案（Residensi Astrea RM874、Kiaramas Danai RM865、Sunway Mont RM938），但這主要是大坪折扣（大坪物業的 PSF 系統性低於中小坪）的正常反映，而非建案品質問題：

說明細節
大坪 PSF 折扣原理RM200 萬 / 2,535 sqft = RM789 psf；若同地段換一個 1,200 sqft 中坪成交 RM150 萬，PSF = RM1,250——同地段、同品質，大坪 PSF 通常低 20–40%
iRumah 分型驗證MK28 1BR/1,514sf = RM890 psf；3BR/2,535sf = RM790 psf——自身內部也顯示大坪折扣 RM100 psf
月租不同步折扣大坪月租（RM9,500–15,000）並非坪數 × 月租 psf——外籍家庭租 3BR 大坪的整體支付意願相對 2BR 中坪並不是 2 倍，但也不是 1 倍。租金效率（月租/入場價）對大坪更有利
Yield 套利結論正是大坪 PSF 折扣 + 月租相對堅挺 = Gross Yield 6.44%——這個 Yield 套利空間是結構性的，除非大坪 PSF 大幅上升（消除折扣）才會消失

1.2 Yield 急升從 5.1%→6.44% 的深層原因分析

假設驗證方式信號
月租上升（積極）若 2024→2025 月租從 RM9,000→RM11,000+，Yield 自然上升若屬實，是持有的最強正面信號
入場 PSF 下降（中性偏負）若 brickz 成交 PSF 下跌，Yield 分母縮小，Yield 數字上升若屬此因，需謹慎
MRT3 前置外籍需求增加MK 外籍家庭提前佈局，租需求上升，推高月租正面且可持續
iProperty YoY PSF +6.7%成交 PSF 上升 +6.7%——如果 PSF 在漲，Yield 上升就純粹來自租金上升，這是最強組合最理想情境——PSF 漲 + Yield 漲 = 雙贏
最強情境判斷：iProperty 顯示 YoY PSF +6.7%，同時 Gross Yield 從 5.1%→6.44%——這意味著月租增速（+26%+）快於 PSF 增速（+6.7%）。這是 MK28 持有者最期待的組合，說明大坪外籍家庭租需求正在強力復甦，且租金定價權回到業主手中。
