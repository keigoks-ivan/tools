§0｜建案定位與第一性原理

A｜Pavilion Square 的核心投資論點

Pavilion Square 的論點核心是：在 KL 黃金三角最後稀缺的 2.02 英畝用地上，直接建於 Pavilion KL Mall（年訪客 3,000萬）上方，以「住在 Pavilion 裡」的步行生活圈體驗 + KL 最長屋頂泳池 + 24 小時禮賓服務，吸引追求終極 Bukit Bintang 步行生活的外籍高淨值人士。

這個論點有三個實質支撐：①地段稀缺性真實（Bukit Bintang 黃金三角 2 英畝土地確實難複製）；②Pavilion KL 直連帶來真實的生活便利性溢價（Pavilion Residences 前身驗證：直連帶動租金 10–15% 溢價）；③Bukit Bintang 步行商業生態圈是 KL 所有地段中最成熟最多元的。

但這個論點同時面臨：①Leasehold + 商業地契雙重折扣；②發售 PSF RM2,300–2,420 已是 Pavilion Residences 成交 RM1,564 的 55% 溢價——這個溢價在 Leasehold 地契下能否在二手市場維持是最大的未解問題；③960 戶中等密度在 Bukit Bintang 竣工後面對大量出租競爭。

第一性問題：RM2,300 psf（Leasehold 商業地契）vs RM1,564 psf（Pavilion Residences，Leasehold 住宅地契）——55% 的溢價有多少是可持續的？

Pavilion Residences 是最直接的歷史對照——同品牌、同地段、同 Leasehold、同直連 Pavilion Mall，16年後成交中位 RM1,564 psf，從發售 RM900 psf 升值 74%，但目前 YoY -2.5%（iProperty 2025）顯示市場在高位整理。若 Pavilion Square 以 RM2,300 psf 入場，即使複製前輩同樣 74% 的 16 年升值，2040 年成交也只有 RM4,002 psf——這是可能的，但代入 Leasehold（比前輩更差的地契結構）的折扣後，升值幅度存疑。

最重要的結構性發現：Pavilion Square（商業地契）比 Pavilion Residences（住宅地契）地契更差，但定價貴 55%。商業地契在馬來西亞市場歷史上一直比住宅地契有折扣——更窄的買家池（無法貸款限制更多）、更高的電費、在 MM2H 框架下合規性更低。Pavilion Square 以更差的地契結構要求更高的 PSF，這個矛盾是本建案估值分析中最需要認真對待的問題。

§1｜市場概況——上游報告框架對位

1.1 上游報告對 Pavilion Square 地段的定性

上游報告框架Pavilion Square 對應信號
「武吉免登 Sky Suites -6.6%（2024）——中端 SR 代表性跌幅」Pavilion Square 在武吉免登，但定位高端（RM2,300 psf）；上游報告的跌幅是「中端SR」，Pavilion Square 嘗試走「頂端差異化」路線，避開這個品類混合——若品牌溢價論點成立則避開；若失敗則面對同樣壓力
「跌的共同點：商業地契 + 高密度量產 + 無差異化」商業地契（-）；中等密度 960 戶（中性）；Pavilion Link Bridge + 品牌差異化（+）——差異化是最大的正面辯護中性偏弱（地契拖累）
「漲的共同點：稀缺性 + 有明確催化劑」稀缺性：黃金三角 2.02 英畝最後土地（真實）；催化劑：Pavilion KL Link Bridge + Bukit Bintang MRT（已運營）正面——稀缺性和催化劑均存在
「武吉免登 SR 積壓是 KL 最嚴重品類」Pavilion Square 在武吉免登，面對系統性積壓壓力；發售均價 RM2,300 是武吉免登 SR 中最高——若無法維持溢價，將面對系統性壓力負面——地段系統性逆風

Pavilion Square 的基建組合是本系列步行生活圈最強的建案——沒有之一。直連 Pavilion KL + Bukit Bintang MRT 5 分鐘 + Jalan Alor 5 分鐘 + Lot 10/Starhill Gallery 步行 + Raja Chulan Monorail——這個組合比 TS2（Imbi Monorail 100m + 天橋直連 Berjaya Times Square）更強，也比 TRX（Exchange TRX 直連）更多元化。對「不叫 Grab」的外籍自住者而言，Pavilion Square 的步行生活體驗在 KL 所有建案中是頂端。

但這個優勢能否支撐 RM2,300 psf Leasehold 商業地契的定價，是整個建案分析的核心懸念。
§12｜買方決策錨點與最終建議
【買方視角】決策框架
最適合者以「Bukit Bintang 步行生活 + 直住 Pavilion KL + 不叫 Grab」為核心需求的外籍高淨值人士；接受 Leasehold 商業地契；視「生活品質 > 資本增值」

Pavilion Square 是本系列分析建案中最兩極分化的案子：基建步行圈（A4/B4）是本系列最高，但幾乎所有其他關鍵維度都顯示進場邏輯薄弱——Leasehold 商業地契（最差地契結構）、Net Yield 1.5–1.9%（本系列最低）、定價 RM2,300 psf 比地契更好的 TRX Residences 更貴、brickz 0 筆成交（無驗證）、武吉免登 SR 系統性積壓逆風。

唯一適合進場的框架：「以步行自住為絕對核心，視 Pavilion KL 直連生活為最高優先，接受 Leasehold 商業地契、接受 Net Yield 低、接受資本增值不確定」——這是一個「購買生活方式，而非投資資產」的框架。在這個框架下，Pavilion Square 是 KL 市場上最純粹的「步行購物中心生活」體驗，無可替代。

但對投資主導框架（資本增值或租金收益）的外籍投資者：不推薦。同樣的 RM115.9 萬預算放在 TRX Residences（Freehold Commercial HDA，brickz YoY +5.56%，MRT 雙線交匯），各方面投資指標都更優。
