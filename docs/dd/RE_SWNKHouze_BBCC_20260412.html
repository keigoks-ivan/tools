🚨 外籍投資者進場前必讀：Leasehold + DBKL 同意限制雙重障礙

① Leasehold（87年，到期 2110年）：本系列八個分析建案中唯一的非 Freehold 建案。Leasehold 對外籍買家的影響：剩餘年限隨時間遞減影響估值；銀行貸款通常對剩餘年限有要求（通常至少 30–40 年以上才能貸款）；二手轉讓時年限遞減會壓低買家願付價格。目前剩餘 84 年（2026竣工至2110年），目前貸款可行，但長期持有後剩餘年限縮短。

② DBKL（Dewan Bandaraya KL）同意限制：官方 HDA 文件明確記載「土地不能在未獲 Dewan Bandaraya Kuala Lumpur 同意的情況下轉讓、出租或變更」——外籍買家除了需要 State Consent，還需要另一層 DBKL 核批，增加轉讓複雜度和時間成本。

③ MM2H Gold 合規問題：MM2H Gold 要求購置 RM100萬+ 物業並持有 10 年。Leasehold 商業地契在 MM2H 持有合規框架下存在不確定性，且 DBKL 同意限制可能影響 MM2H 審批。強烈建議事前向律師確認。

總結：SWNK Houze 對外籍投資者而言，Leasehold + 商業地契 + DBKL 同意限制是三重不利因素。若以 MM2H 合規為目的，不建議選擇此建案。

§0｜建案定位與第一性原理

A｜SWNK Houze 解決什麼需求？——與 TS2 武吉免登的比較框架

SWNK Houze 的投資論點核心是 BBCC 整合開發的 TOD（Transit-Oriented Development）效應——三站捷運直連、LaLaport 1 分鐘步行、Gourmet Street、Zepp Hall、BBCC Phase 3 Signature Tower 持續升值，形成一個「越成熟越有價值」的整合生態圈。

這個論點有其合理性，但有一個關鍵問題：Leasehold 商業地契。BBCC 的 TOD 生態圈確實成熟，但這個價值不能以 Freehold 的方式持有——每一年的持有，剩餘年限就少一年，理論上對外籍長期投資者而言，資產的「真實」年限遞減是財務上無法忽視的隱性成本。

以 KL RM1M+ 深度報告的框架對照：SWNK Houze 最類似「TS2 武吉免登」的定位——功能性 TOD 建案，商業地契，適合「偶爾自住 + 短租 + 接受增值弱」的持有邏輯，但 SWNK Houze 更多加了 Leasehold 的一層結構性折扣。

關鍵定位問題：SWNK Houze vs TS2 武吉免登

TS2 武吉免登：商業地契（Freehold）+ 較高功能性（天橋直連 Berjaya Times Square）+ RM100萬+ MM2H 合規
SWNK Houze：Leasehold 商業地契 + 更強的 TOD 連接（三站 vs Monorail 一站）+ LaLaport 生態圈 + 起步價更低（RM74.5萬）

若已持有 TS2，SWNK Houze 在功能定位上高度重疊但地契更弱。對現有 TS2 持有者而言，SWNK Houze 提供的增量價值有限。
