§0｜建案定位與第一性原理

A｜Kiaramas deDaun 的核心投資論點

Kiaramas deDaun 的投資論點是本系列中邏輯最清晰的：在 KL 增速最高的外籍聚居地段（Mont Kiara YoY +6.4%），以「Freehold 住宅地契 + 低密度 + 知名精品系列開發商（Asia Quest，第六個 Kiaramas）+ 最低管理費（RM0.36 psf）+ 本系列最強銷售紀錄（97% 取購率）」的組合，定價 RM844–918 psf，比直接對照組 Kiaramas Danai（同開發商，鄰接，成交中位 RM865 psf）幾乎持平——即用「等同二手成交 PSF 的價格買入新案期房」。

🟢 本系列最強核心發現：Kiaramas deDaun 的發售 PSF（RM844–918）與同開發商鄰接已完工建案 Kiaramas Danai 的 brickz 成交中位（RM865）幾乎完全一致——這意味著買家以「二手市場驗證價」買入新建案，在同等品質 Freehold 住宅地契的框架下，幾乎沒有任何定價溢價風險。加上 Phase 2 定價微漲至 RM918 psf，形成正面錨點。這個定價邏輯在本系列所有建案中是最穩健的。
