# HUNTR/X：魂門之戰 — 遊戲設計稿 v1

> 狀態：設計階段，尚未實作。2026-07-17。
> 部署目標：myproperty.investmquest.com/game/（靜態 Cloudflare Pages，純前端）

## 一、定位

無雙割草類（海賊無雙式）2.5D 俯視動作遊戲。主角為《KPop 獵魔女團》三位主角（Rumi／Mira／Zoey），開場選角，先做一個關卡。同人自製、非商用：所有美術用自繪像素／幾何風格，配色致敬但不使用電影原始素材、不用電影音樂，避免版權問題。

## 二、技術架構（受網站限制決定）

- 純前端：`/game/index.html` ＋ `game.js`（可拆 `entities.js`、`stage1.js`），零外部依賴、零 build step。
- Canvas 2D，固定邏輯解析度 960×540，CSS 等比縮放適配手機。
- 俯視斜角（2.5D）視角：一張大地圖、鏡頭跟隨主角——這是最省成本又最有「無雙同屏百人」感的做法。
- 目標 60fps、同屏 60～80 隻小兵：物件池＋簡化碰撞（圓形 hitbox）＋離屏敵人降頻 AI。
- 操作：鍵盤（WASD／方向鍵移動，J 普攻、K 重攻、L 必殺）＋手機虛擬搖桿與三顆按鈕。
- 存檔：localStorage 記最高評價與解鎖。

## 五、第一關：「首爾夜市・魂門裂縫」

一張長條形夜市大地圖（約 3 屏寬），從街口一路推進到裂縫廣場。流程約 5～8 分鐘。

```
[入口街道]──→[夜市主街]──→[十字路口]──→[裂縫廣場(Boss)]
  Zone A        Zone B         Zone C          Zone D
```

- **Zone A 入口街道（教學波）**：15 隻小鬼分兩波。牆上浮字教操作（移動／普攻／連段）。無壓力，讓玩家爽 30 秒。
- **Zone B 夜市主街（割草主菜）**：連續 3 波共 45 隻（小鬼＋擲矛鬼混編），第 3 波加入 2 隻盾鬼並浮字提示重攻擊。攤位可破壞、掉補給。清完開門。
- **Zone C 十字路口（精英戰）**：2 隻陰差精英＋週期性小鬼增援（每 15 秒 6 隻，殺精英即停）。教玩家「先斬將、雜兵自潰」的無雙節奏。
- **Zone D 裂縫廣場（Boss）**：陰差隊長，圓形競技場，入場封路。

節奏曲線：爽（A）→ 更爽（B）→ 緊張（C）→ 高潮（D）。必殺表設計上讓玩家在 B 尾與 D 各能放一次大招。

## 六、Boss：陰差隊長（兩階段）

- **P1（HP 100%→50%）**：三連斬（近身）、直線衝鋒（有 0.8 秒紅線預警）、召喚 6 隻小鬼。攻擊間隙大，教玩家看預警走位。
- **P2（HP 50%→0）**：怒吼變身（紫焰特效），攻速 +30%，新增「環形音波」（以 Boss 為圓心的擴散圈，需跳離／走位），召喚改 10 隻。
- 擊破演出：慢動作最後一擊＋畫面閃白＋結算。

## 七、結算與重玩性

- 結算畫面：擊殺數／最大連段／通關時間／受傷次數 → S/A/B/C 評價。
- S 條件範例:擊殺 ≥ 100、死亡 0、時間 ≤ 6 分。
- 三角色各打一次同關卡手感完全不同＝內建三倍重玩價值。最高評價按角色分別記錄（localStorage）。

## 八、畫面與音效

- 場景：夜市霓虹（深藍底＋霓虹粉紫招牌）、地面斜角磁磚、裂縫處紫黑漩渦。全部 Canvas 繪製或小型自製 tileset。
- 打擊感三件套：hit-stop（命中凍 2 幀）、擊退位移、白閃受擊。割草爽感的核心，優先級最高。
- 音效：Web Audio API 程式合成（斬擊／命中／必殺／Boss 吼），不引外部音檔；BGM 可先無或用合成 loop。

## 九、實作切分（之後動工用）

1. M1 核心手感：移動＋普攻連段＋小鬼＋hit-stop（先只做 Rumi）
2. M2 波次系統＋Zone A/B＋掉落物
3. M3 另外兩角色＋選角畫面
4. M4 盾鬼／精英＋Zone C
5. M5 Boss＋結算＋評價
6. M6 手機觸控＋音效＋首頁入口連結

每個 M 都是可玩狀態，隨時可以 push 上線試玩。
